    PDF 편집기 사용 가이드

    복잡한 설치형 소프트웨어 없이도 브라우저에서 즉시 PDF 문서를 병합하거나 분할할 수 있습니다. 본 도구는 사용자의 개인정보 보호를 위해 모든 작업을 로컬 브라우저 내에서만 처리합니다.

    기능별 사용 방법

    • PDF 합치기 (Merge): 여러 개의 PDF 파일을 업로드한 뒤, 리스트에서 순서를 조정하여 하나의 완성된 문서로 만듭니다.
    • PDF 나누기 (Split): 방대한 PDF 문서에서 필요한 페이지만 쏙 골라내어 별도의 파일로 저장할 수 있습니다.
    • 페이지 범위 지정: 나누기 기능 사용 시 '1, 3, 5-10'과 같은 직관적인 방식으로 추출할 페이지를 선택하세요.

    자주 묻는 질문 (FAQ)

    Q. 한 번에 합칠 수 있는 파일 수에 제한이 있나요?

    기술적으로는 제한이 없으나, 브라우저의 메모리 성능에 따라 10~20개 이상의 대용량 파일을 한 번에 처리할 경우 다소 시간이 걸릴 수 있습니다.

    Q. 암호가 걸린 PDF 파일도 편집 가능한가요?

    보안상의 이유로 암호(비밀번호)가 걸려 있는 PDF 파일은 편집이 불가능합니다. 먼저 암호를 해제한 후 업로드하여 사용해 주시기 바랍니다.
